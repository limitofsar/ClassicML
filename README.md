# ClassicML

Библиотека для классического машинного обучения на Python.  
Содержит основные модели регрессии, классификации, ансамбли, кластеризации и методы снижения размерности.

---

## Содержание проекта

### Готовые модели (в ветке `main`)
- `Linear_regression` — линейная регрессия
- `Logistic_Regression` — логистическая регрессия
- `SVM` — метод опорных векторов
- `Knn_Classifier` — KNN классификация
- `Knn_Regression` — KNN регрессия

### Планируемые модели (в ветке `develop`)
#### Деревья решений
- Дерево решений (классификация)
- Дерево решений (регрессия)

#### Ансамбли
- Случайный лес (регрессия)
- Случайный лес (классификация)
- Бэггинг (регрессия)
- Бэггинг (классификация)
- Градиентный бустинг (регрессия)
- Градиентный бустинг (классификация)

#### Кластеризация
- K-Means
- Иерархическая агломеративная кластеризация
- DBSCAN

#### Снижение размерности
- PCA (Метод главных компонент)
